# MACS — Multi-Agent Collaboration System

> A research specification for **OS-native, recursive agent orchestration** — a society of LLMs built on process spawning, filesystem memory, and shell-level coordination instead of framework abstractions.

![Research](https://img.shields.io/badge/Type-Research%20Spec-0ea5e9?style=for-the-badge)
![Multi-Agent](https://img.shields.io/badge/Multi--Agent-Architecture-8b5cf6?style=for-the-badge)
![Biomimetic](https://img.shields.io/badge/Approach-Biomimetic-22c55e?style=for-the-badge)

MACS argues for treating LLMs as biological entities and constructing a **hierarchical society** around them using OS primitives — rather than building abstraction layers over models the way LangChain, AutoGen, or CrewAI do.

## Core ideas

- **OS-native orchestration** — process spawning, filesystem-based shared memory, shell-level coordination.
- **Semantic Mixture of Experts (SMoE)** — routing that is explicit, auditable, and steerable through natural-language reasoning instead of opaque math.
- **Biomimetic architecture** — mirrors specialized cortical regions operating in parallel under executive (prefrontal) coordination, enabling emergent intelligence through orchestration.
